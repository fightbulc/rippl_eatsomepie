Rippl Eat Some Pie

The following code:

DOM element:
<div id="circle"></div>

JS magic:
EatSomePie({target:'circle', eaten:50, speed:500});
EatSomePie({target:'circle', eaten:10, label: true});
EatSomePie({target:'circle', eaten:33, colorEaten:'#c00', label: true});
EatSomePie({target:'circle', radius:100, eaten:63, label:'3h left', speed:5000});

Results in:

Options:

{
target: ElementID (string)
eaten: Percentage of eaten pie (number)
radius: size of pie (number / optional)
colorBase: hex background color (string / optional)
colorEaten: hex background color (string / optional)
speed: animate pie (MS number / optional)
label: true/false/String (bool,string / optional)
labelSize: text size (number / optional)
labelColor: hex text color (string / optional)
}
